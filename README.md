# Fairoz Portfolio 2026

A modern, high-performance personal portfolio website built with the latest web technologies. This project showcases professional experience, projects, and skills through an interactive and accessible user interface.

## 🚀 Overview

This application is engineered to provide a seamless user experience with smooth animations, responsive design, and robust accessibility features. It serves as a digital resume and showcase for software development work.

## ✨ Features

- **Modern Architecture**: Built on React 19 and TypeScript for type-safe, maintainable code.
- **High Performance**: Powered by Vite for lightning-fast development and optimized production builds.
- **Interactive UI**:
  - **Animations**: Complex transitions and micro-interactions using Framer Motion and GSAP.
  - **Smooth Scrolling**: Integrated Lenis for a premium scrolling feel.
  - **Grain Effect**: Custom texturing for visual depth.
- **Responsive Design**: Mobile-first approach ensuring perfect rendering across all device sizes.
- **Dynamic Routing**: Client-side routing with React Router DOM v7, including detailed experience pages.
- **Form Handling**: Robust form validation using React Hook Form and Zod.
- **Component System**: Modular UI architecture utilizing Radix UI primitives for accessibility.

## 🛠️ Tech Stack

### Core
- **React 19**: UI Library
- **TypeScript**: Static Typing
- **Vite**: Build Tool & Bundler

### Styling & UI
- **Tailwind CSS**: Utility-first CSS framework
- **Radix UI**: Headless UI components for accessibility
- **Lucide React**: Iconography
- **Class Variance Authority (CVA)**: Component variant management

### Animation & Interaction
- **Framer Motion**: Gesture and layout animations
- **GSAP**: Advanced sequencing and effects
- **Lenis**: Smooth scroll standardization

### State & Logic
- **React Router DOM v7**: Navigation and routing
- **React Hook Form**: Form state management
- **Zod**: Schema validation
- **Date-fns**: Date manipulation

## 🏁 Getting Started

Follow these steps to set up the project locally.

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Fairoz007/fairoz.in-2026.git
    cd fairoz.in-2026
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

4.  Open your browser and navigate to `http://localhost:5173`.

## 📜 Scripts

- `npm run dev`: Starts the development server with HMR.
- `npm run build`: Compiles the application for production using TypeScript and Vite.
- `npm run lint`: Runs ESLint to ensure code quality and consistency.
- `npm run preview`: Locally previews the production build.
